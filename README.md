# red-wine-data-science

🍷 Red Wine Quality Prediction (Kırmızı Şarap Kalite Tahmini)
Bu proje, kırmızı şarapların kimyasal özelliklerini kullanarak kalite puanlarını tahmin eden bir makine öğrenmesi sınıflandırma projesidir.

🚀 Proje Hakkında
Bu çalışmada, [buraya veri setini nereden aldıysan linkini koy, örneğin: Kaggle] üzerinde bulunan kırmızı şarap veri seti kullanılmıştır. Amacım, şarabın kimyasal bileşenlerinin (alkol, pH, asitlik vb.) kalite üzerindeki etkisini istatistiksel olarak incelemek ve bir sınıflandırma modeli ile şarabın "iyi" veya "orta/düşük" kalitede olduğunu tahmin etmektir.

🛠 Kullanılan Teknolojiler & Kütüphaneler
Python

Pandas (Veri manipülasyonu)

Matplotlib & Seaborn (Veri görselleştirme)

Scikit-learn (Makine öğrenmesi)

📊 Önemli Bulgular (EDA)
Alkol Oranı: Şarap kalitesini etkileyen en önemli faktörün alkol oranı olduğu tespit edilmiştir.

Korelasyon: Uçucu asitlik (volatile acidity) ile kalite arasında negatif bir ilişki olduğu gözlemlenmiştir.

Model Performansı: Random Forest algoritması kullanılarak %79 doğruluk (accuracy) oranına ulaşılmıştır.
